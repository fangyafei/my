mac10.9开发环境配置

作者：admin 经验之谈 Tags：memcache mac mysql brew 2014-10-20 13:35:26 来源：未知 浏览：

原文地址：

#apache httpd.conf 中需要设置以下内容
<Directory />
    Options FollowSymLinks
    #AllowOverride None
    AllowOverride all
    Order deny,allow
    allow from all
</Directory>
 
 
前提条件（xcode需要安装）：
sudo ln -s /Applications/Xcode6-Beta.app/Contents/Developer/Platforms/MacOSX.platform/Developer/SDKs/MacOSX10.10.sdk/usr/include /usr/include 
 
brew安装:
curl -LsSf http://github.com/mxcl/homebrew/tarball/master | sudo tar xvz -C/usr/local --strip 1      
 
解决 mcrypt 模块不存在：
 
wget  http://jaist.dl.sourceforge.net/project/mcrypt/MCrypt/2.6.8/mcrypt-2.6.8.tar.gz
tar zxvf libmcrypt-2.6.8.tar.gz
cd libmcrypt-2.6.8
/configure --disable-posix-threads --enable-static
make
sudo make install
cd..
 
wget http://219.239.26.4/files/1124000003A6741E/cn2.php.net/distributions/php-5.5.14.tar.gz
tar zxvf php-5.5.14.tar.gz
cd php-5.5.14/ext/mcrypt
phpize
./configure
make
sudo make install
#sudo cp modules/*.so /usr/lib/php/extensions/no-debug-non-zts-20121212/  
 
memcached:
wget http://pecl.php.net/get/memcache-2.2.0.tgz
tar zxvf  memcache-2.2.0.tgz
cd memcache-2.2.0
phpize
./configure
make
sudo make install
#sudo cp modules/*.so /usr/lib/php/extensions/no-debug-non-zts-20121212/   
 
redis:
wget http://pecl.php.net/get/redis-2.2.5.tgz
tar zxvf  redis-2.2.5.tgz
cd redis-2.2.5
phpize
./configure
make
sudo make install
#sudo cp modules/*.so /usr/lib/php/extensions/no-debug-non-zts-20121212/   
 
mongodb:
wget http://pecl.php.net/get/mongo-1.5.4.tgz
tar zxvf mongo-1.5.4.tgz
cd mongo-1.5.4
phpize
./configure
sudo make install
#sudo cp modules/*.so /usr/lib/php/extensions/no-debug-non-zts-20121212/    
 
sudo vi /etc/php.ini
#（如果不存在此文件：sudo cp /etc/php.ini.default /etc/php.ini）
#在文件最后加上：
extension_dir = "/opt/php/lib/php/extensions/no-debug-non-zts-20100525/"
 
 
#重启apache
sudo apachectl stop
sudo apachectl start 
 
#
mongod --dbpath=/Users/sunxun/data/mongodb/datas --logpath=/Users/sunxun/data/mongodb/logs/production.log --port 27018 --logappend --fork
memcached &
redis-server &
 
mysql 在系统设置偏好里进行启动
 
#安装 phpRedisAdmin
git clone https://github.com/ErikDubbelboer/phpRedisAdmin.git
cd phpRedisAdmin
git clone https://github.com/nrk/predis.git vendor
cd includes
cp config.sample.inc.php config.inc.php
vi config.inc.php
