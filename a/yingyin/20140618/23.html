度科幻电影一：《宝莱坞机器人之恋》

　　主演：艾西瓦娅·雷，拉吉尼坎塔

　　类型：喜剧 / 动作 / 爱情 / 科幻

　   上映日期：2010-10-01

　   宝莱坞科幻大片，讲述一位天才科学家设计的智能机器人"七弟"他有了人类的感情，爱上了这个科学家的女朋友。正因为如此，导致机器人“七弟”后来被科学家的老师利用，变成了杀人机器。  
      最终科学家取出安放在机器人体内的芯片，使他回到了以前那个善良的“七弟”，在审判席，法官命令科学家摧毁机器人，科学家不忍心下手，“七弟”进行了自我解体。

印度科幻电影二：《冻结的幸福》

　　主演：尼尔·尼汀·穆科什 / 碧帕莎·芭素 / 索菲亚·查德哈瑞 /Rahul Dev/Bobby Vatsa

　　类型：科幻

　　雷尼尔Nitin生物(Mukesh),都要为他的摄影师……直到他继承一个非常的特别的照相机,他从他的祖父在某种程度上改变生活,他也不能想象他喜出望外。“权力”的改变命运的射线照相机之间。他的生活变成一个大坐过山车,他发大财并帮助他遇见了他的生活,Simi爱》)(Bipasha是一件令人兴奋的性感的DJ和一个有主见的人。每件事都是走向“肥多利,从此过上了幸福的。正如他们所说的能力越大,责任和光线的情况,也很危险。他现在必须面对的现实和赢得不仅与邪恶的力量,而且攻击的命运。爱立信是Aa Dekhen惊悚片,浪漫的音乐与一些伟大的边缘的座位行动。可以改变他的日期与命运的光线吗?

印度科幻电影三：《天使战将》

　　主演: 沙鲁克·汗 / 卡琳娜·卡普 / 阿俊·拉姆鲍 / Tom Wu / Shahana Goswami / Satish Shah / 达里普·塔西尔

　　类型: 动作 / 科幻

　　上映日期: 2011-10-26(印度)

　　电影剧情描述一位父亲Sherhar是伦敦龙头电玩公司的程式设计师，為了打进儿子Prateek的酷炫世界，Sherhar创造出一个永远不会被打死的反派电玩角色Ra One;并按照自己的样子打造一个负责追捕Ra One的超级英雄G One。但随著游戏开始进行， 一切却渐渐失控走样，Ra One不仅逃出游戏页面来到现实世界，还大肆胡作非為;游戏製作团队这才知道事态严重，而唯一的解决之道，就只有想办法释放电玩中的英雄G One同样来到现实世界，并找到Ra One的致命关键弱点，一举击败他!原本关系疏离的父子俩人，将為了同一任务一起并肩作战，拯救地球;但难缠的Ra One会如此轻易的放过他们吗?...


 

印度科幻电影四：《功夫小蝇》

　　主演: Nani / Samantha Ruth Prabhu / Sudeep

　　类型: 剧情 / 科幻 / 奇幻

　　上映日期: 2012-07-05

　　纳尼与邻楼的女孩宾都相爱，但他们从未对彼此述说衷肠，两人继续享受着心动的默恋。哪知一个大老板苏迪普也看上了她，这个看似腰缠万贯、有权有势的男人实则是个心狠手辣的黑帮老大。他用尽手段欲引诱宾都，但他发现宾都爱的是纳尼。

　　苏迪普为了得到宾都，在一天晚上，将纳尼杀死。纳尼死时知道苏迪普为了宾都，告诉苏迪普，“离宾都远点，否则，我杀了你”!

　　死后的纳尼重生为一只苍蝇，开始对苏迪普报复。最后，宾都知道了纳尼的死因，也知道了眼前这只苍蝇便是纳尼。两人便联合起来。宾都将纳尼全副武装，对苏迪普实施轮番攻击，苏迪普为了防御这只苍蝇，费劲了精力。当他知道宾都在暗中帮助这只苍蝇后，将宾都抓来，试图将她和纳尼一起杀死。小小苍蝇的报复，结局会是怎样呢?

　　影片笑料不断，3D效果也很特别。利用苍蝇的眼睛来看一切。值得一看!

印度科幻电影五：《印度超人》

　　主演: Sharat Saxena / 朴雅卡·乔普拉

　　类型: 动作 / 爱情 / 科幻 / 冒险

　　上映日期: 2006-06-23

　　《卡里什》讲述罗希特的儿子卡里沙的故事，卡里沙继承了父亲的所有超能力，甚至青出于蓝。然而在他印度田园诗一样的生活中，卡里沙并没有发现他的这些超能力，直到和心上人、一个电视记者一起去新加坡的时候遇见了一个疯狂科学家，他发明的一项技术将会永远地改变世界。卡里沙终于变身为超级英雄卡里什，阻止了科学家的阴谋。
